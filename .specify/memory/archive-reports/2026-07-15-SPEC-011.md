SPEC-011 shipped deterministic, opt-in execution-flow and functional-cluster
catalogs. Flow analysis detects statically registered entry points, traces bounded
call graphs with per-step provenance and explicit truncation state, and exposes
paged summaries plus flow detail over MCP and REST. Cluster analysis builds a
deterministic file graph, computes Louvain clusters, preserves stable identities
across re-indexes, and stores heuristic labels with optional LLM display labels
kept presentation-only. Both catalogs recompute after successful index/sync and
fail independently without failing core indexing. tasks.md closed 68/68.
