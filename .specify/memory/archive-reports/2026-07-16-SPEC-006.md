## Summary

SPEC-006 shipped the local-first web graph browser served by
`codegraph serve --web`. The Vite/React/Tailwind/shadcn application provides
repository status and switching, search and symbol detail, callers/callees and
catalog views, a Cytoscape graph browser, impact views, re-analysis progress,
and same-origin chat routed through the SPEC-018 backend boundary. Static
assets ship inside the npm package and the browser makes no CDN or direct
provider calls. All 96 implementation tasks were complete before merge.
